Easy Pirate TurnBased Game

Choose your player and fight for Tortuga!
step 1:
chose your team from left or right side
step 2
pick your player
step 3
computer picks its player from opposite side
step 4
you attack
step 5
computer attacks
step 6
the first one to die loses
step 7
Game Over



Libraries Used:

JavaScript, jQuery

How to use it:

Clone my repo
Run npm install
Run bower install
